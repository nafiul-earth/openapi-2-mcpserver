# Setup the Projects

As the new API defintion is a bit a different than old version so lets try also the OpenAPI defintion for watsonx

1. Intialize the projects

```
uv init
```

2. Initialize the Virtual Environment 

```
uv venv
```

3. Activate the Virtual Environment 

```
source .venv/bin/activate
```
4. Add necessary dependencies
```
uv add httpx fastapi dotenv uvicorn
```


5.  Get all the api documentation url for different watsonx services
   
   ```
   API_DEFINITIONS = [
  {
    "name":"watsonx-ai-cp",

  },
  {
    "name":"watsonx-ai",

    "name":"machine-learning-cp",

    "version":"5.1.2"
  },
  {
    "name":"ai-openscale",
    "url":"https://cloud.ibm.com/apidocs/ai-openscale.json"
  },
  {
   "name":"ai-openscale",

  }

]
```
7. Once we have the API defitnion, run the code to generate the tools 
   
```
uv run main.py
```

8. Invoke the tools api in browser 
   ```
   http://localhost:8000/tools
   ```
